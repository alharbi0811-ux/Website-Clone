أريدك أن تنشئ نظام إدارة حالات الفئات (Category Status System) داخل موقعي بحيث يمكن التحكم بحالة كل فئة (مفتوحة / مغلقة / قيد العمل) مع صلاحيات عرض مختلفة حسب نوع المستخدم (Admin / Player).

━━━━━━━━━━━━━━━━━━━━━━━
🔹 الهدف:

إدارة ظهور الفئات داخل الموقع بحيث:
- بعض الفئات تكون مفتوحة وجاهزة للاستخدام
- بعض الفئات تكون مغلقة مؤقتًا
- بعض الفئات تظهر قيد العمل
- مع اختلاف العرض بين الأدمن واللاعب

━━━━━━━━━━━━━━━━━━━━━━━
🔹 الفئات المفتوحة دائمًا (لا يتم إغلاقها افتراضيًا):

- ولا كلمة
- دول و عواصم
- قصص الانبياء
- اسلامي
- حروف
- معلومات عامة
- الكويت

هذه الفئات تكون:
- is_locked = false بشكل دائم افتراضيًا
- يمكن للأدمن تعديلها يدويًا إذا أراد

━━━━━━━━━━━━━━━━━━━━━━━
🔹 باقي الفئات:

يمكن أن تكون:
- مفتوحة
- مغلقة
- قيد العمل

━━━━━━━━━━━━━━━━━━━━━━━
🔹 قاعدة البيانات:

تعديل جدول categories:

- id
- name
- slug
- status (open | closed | in_progress)
- lock_message (nullable)
- is_default_open (boolean)

━━━━━━━━━━━━━━━━━━━━━━━
🔹 منطق العرض للمستخدم (Player):

1. إذا status = open:
   → تظهر الفئة بشكل طبيعي

2. إذا status = closed:
   → لا يمكن الدخول إليها
   → تظهر بطاقة الفئة مع:
     • طبقة تغطية (overlay)
     • نص: "مغلقة"

3. إذا status = in_progress:
   → تظهر لكن غير قابلة للدخول
   → يظهر نص:
     "جار العمل عليها"

━━━━━━━━━━━━━━━━━━━━━━━
🔹 ملاحظة مهمة لللاعب:

- لا يمكنه الدخول للفئات المغلقة أو قيد العمل
- فقط يرى الحالة

━━━━━━━━━━━━━━━━━━━━━━━
🔹 منطق العرض للأدمن (Admin):

الأدمن لديه صلاحيات كاملة:

- يمكنه فتح أي فئة
- يمكنه إغلاق أي فئة
- يمكنه وضعها قيد العمل

━━━━━━━━━━━━━━━━━━━━━━━
🔹 عرض خاص للأدمن:

إذا كانت الفئة:
- closed أو in_progress

يظهر للأدمن داخل البطاقة:

- مربع متوسط الحجم باللون الأحمر
- نص داخله: "مغلق"

لكن:
- الأدمن يستطيع الضغط عليها والدخول لها بشكل طبيعي
- لإدارة بنك الأسئلة وتعديلها

━━━━━━━━━━━━━━━━━━━━━━━
🔹 لوحة التحكم:

إضافة قسم داخل إدارة الفئات:

"حالة الفئة"

ويحتوي على:

- Toggle: فتح / إغلاق / قيد العمل
- زر حفظ الحالة
- خيار كتابة رسالة مخصصة (lock_message)

━━━━━━━━━━━━━━━━━━━━━━━
🔹 تحسينات العرض:

- overlay شفاف فوق الفئات المغلقة
- animation خفيفة عند الإغلاق
- badge صغير يوضح الحالة:
  • Open = أخضر
  • Closed = أحمر
  • In Progress = أصفر

━━━━━━━━━━━━━━━━━━━━━━━
🔹 الهدف النهائي:

نظام إدارة فئات احترافي يسمح بـ:

- التحكم الكامل بحالة كل فئة
- إظهار محتوى مختلف للاعب والأدمن
- حماية الفئات غير الجاهزة من الاستخدام
- مع إبقاء الأدمن قادر على إدارتها بالكامل
